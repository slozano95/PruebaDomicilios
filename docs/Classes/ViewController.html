  • getDatos()

    getDatos

    Mediante Alamofire se genera la petición get a la url definida en Helper, se espera recibir texto en formato JSON, se decodifica utilizando SwiftyJSON y se carga en memoria local, se recarga la tabla para mostrar los datos si toda la operación fue exitosa. Se utiliza SwiftSpinner para mostrar views de carga de datos y errores, permite hacer mejor la UX.

    Requires

    Alamofire, SwiftyJSON, SwiftySpinner
