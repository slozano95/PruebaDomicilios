    getDatos

    Mediante Alamofire se genera la petición get a la url obtenida desde el objeto recibido por el segue invocado en la tabla de ViewControllerse espera recibir texto en formato JSON, se decodifica utilizando SwiftyJSON y se carga en memoria local. Se utiliza SwiftSpinner para mostrar views de carga de datos y errores, permite hacer mejor la UX.
